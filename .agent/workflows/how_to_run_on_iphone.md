# 아이폰에서 앱 실행하기

이 앱을 아이폰에서 실행하는 방법은 크게 두 가지가 있습니다.

## 1. 같은 와이파이에서 실행 (개발용)
PC와 아이폰이 **같은 와이파이**에 연결되어 있어야 합니다.

1. PC에서 터미널을 열고 `npm run dev`가 실행 중인지 확인합니다.
2. 터미널에 다음과 같은 주소가 표시될 것입니다:
   ```
   > Network:  http://192.168.x.x:5173/
   ```
3. 아이폰 사파리(Safari) 브라우저를 켜고, 위 주소(예: `http://192.168.0.10:5173`)를 그대로 입력합니다.
4. 앱이 실행됩니다!
   - **팁**: 사파리 하단 공유 버튼 -> '홈 화면에 추가'를 누르면 진짜 앱처럼 아이콘이 생깁니다.

## 2. 인터넷에 배포하기 (추천)
어디서든 접속하려면 무료 호스팅 서비스인 **Vercel**을 사용하는 것이 가장 좋습니다.

1. GitHub에 코드를 올립니다.
2. [Vercel](https://vercel.com)에 가입하고 GitHub 레포지토리를 연결합니다.
3. 배포가 완료되면 제공되는 URL(예: `https://bodyapp.vercel.app`)로 접속하면 됩니다.

## 📱 진짜 앱처럼 아이콘 만들기 (PWA)
이 앱은 **PWA(Progressive Web App)** 기술이 적용되어 있어, 설치하면 앱처럼 쓸 수 있습니다.

1. 아이폰 사파리에서 앱 주소로 접속합니다.
2. 하단 중앙의 **공유 버튼** (네모에서 화살표 나가는 아이콘)을 누릅니다.
3. 메뉴를 내려서 **'홈 화면에 추가'**를 누릅니다.
4. 홈 화면에 아이콘이 생깁니다. 이제 이걸 누르면 주소창 없이 **진짜 앱처럼 전체 화면으로 실행**됩니다!

> **참고**: 현재 방식은 "웹 앱"입니다. 앱스토어에 올리는 "네이티브 앱"을 만들려면 Mac 컴퓨터와 Xcode, 그리고 개발자 계정($99/년)이 필요합니다. 현재 환경에서는 이 PWA 방식이 가장 효율적이고 빠릅니다.

## ❓ 자주 묻는 질문

### Q. 컴퓨터를 꺼도 앱이 되나요?
**1번 방법(와이파이 연결)**을 쓰신다면 **컴퓨터가 켜져 있어야 합니다.** 컴퓨터가 서버 역할을 하기 때문입니다.
컴퓨터를 꺼도 쓰고 싶다면 **2번 방법(인터넷 배포)**을 이용해야 합니다. Vercel 같은 무료 호스팅 서비스를 이용하면 컴퓨터를 꺼도 24시간 접속 가능합니다.

### Q. 인터넷 배포는 어떻게 하나요?
가장 쉬운 방법은 **Vercel**을 이용하는 것입니다.
1. [GitHub](https://github.com)에 회원가입하고 이 코드를 업로드합니다.
2. [Vercel](https://vercel.com)에 가서 'Add New Project'를 누르고 GitHub의 코드를 불러옵니다.
3. 'Deploy' 버튼만 누르면 끝! 이제 전 세계 어디서든 접속 가능한 주소가 생깁니다.
